# 📊 Data Professional Survey Dashboard — Power BI

## Overview
An interactive Power BI dashboard analysing survey responses 
from 600+ data professionals worldwide, built as part of the 
Alex The Analyst Power BI for Beginners course.

## Key Insights
- Average salary breakdown by job title and country
- Most popular programming languages among data professionals
- Work-life balance and salary satisfaction scores

## Tools Used
- Power BI Desktop
- Power Query (data cleaning & transformation)

## Data Source
Survey data provided as part of Alex The Analyst's 
Power BI tutorial series on YouTube.
